# Marketing

This folder contains blog posts, social media content, and other marketing material to promote nbkp (Nomad Backup) -- an rsync-based backup tool for nomadic setups with sentinel-guarded volumes and optional btrfs/hard-link snapshots.

- [Intro Blog Post](./r-backup/intro-post.md): An introduction to nbkp, its features, and use cases, written for the [r/Backup](https://www.reddit.com/r/Backup/) Subreddit.
